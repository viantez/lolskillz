# CLAUDE.md — Viantez Skills Repo

## Structure

Skills are organized into bucket folders under `skills/`:
- `productivity/` — Coaching workflow tools

Every skill in `productivity/` must:
1. Have a reference in `README.md` with the skill name linked to its `SKILL.md`
2. Have an entry in `.claude-plugin/plugin.json`

Each SKILL.md must have frontmatter with `name` and `description`.

## Language

| Term | Definition |
|------|-----------|
| **Schüler** | The person being coached |
| **Session** | A coaching unit (1-3h) |
| **Diagnostic** | Analysis phase before intervention |
| **Eskrima** | 5-level mindset framework |
| **Forge of Legends** | The coaching platform |

See `skill-context` for the full glossary.

## Naming Convention

Skills in this repo follow `skill-<name>` pattern (lowercase, hyphens).
- Skill directories go under `skills/productivity/<skill-name>/`
- Each contains `SKILL.md`

## No personal skills

Skills in this repo are public and shared. Do not add repo-specific or personal skills here.
